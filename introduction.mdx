## Get answers to quiz questions instantly

Screen Answerer is an AI-powered application that helps you find answers to quiz questions in real-time. Using Google Gemini AI, it can automatically detect questions from your screen or process text and image-based questions you submit directly.

## How it works

<Steps>
  <Step title="Set up your API key">
    Get a free Google Gemini API key and configure it in the application settings. Your key is stored securely in your browser's local storage.
  </Step>
  <Step title="Choose your input method">
    Either enable real-time screen monitoring to automatically detect questions, or submit questions directly via text or image upload.
  </Step>
  <Step title="Get instant answers">
    Screen Answerer processes your question using Google Gemini AI and displays the answer immediately, with support for formatted markdown responses.
  </Step>
</Steps>

<Info>
  Screen Answerer uses the Google Gemini API, which requires a free API key. The application runs entirely on your local machine and uses your own API quota.
</Info>

Provide instant assistance for users who need help understanding quiz questions in real-time.

## Technology stack

- **Backend**: Node.js with Express server
- **Frontend**: HTML, CSS, and vanilla JavaScript
- **AI**: Google Gemini 2.0 Flash and Flash Lite models
- **Security**: Helmet.js middleware and rate limiting
- **File handling**: Multer for image uploads

<Note>
  Screen Answerer is designed as a local-first application. All processing happens on your machine using your own Gemini API key.
</Note>
